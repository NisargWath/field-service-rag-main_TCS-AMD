You are helping me continue a hackathon project.

Project Name:
Multi-Modal AI Field Service Assistant for Energy & Utility Field Technicians

Core Idea:
A field technician can ask a text question, speak a question hands-free, upload an equipment photo, or scan a warning label/nameplate. The system retrieves grounded safety and maintenance guidance from official manuals and returns a numbered technician checklist with source citations.

The project runs on AMD Developer Cloud through JupyterLab.

CURRENT SYSTEM ARCHITECTURE

Frontend:

- Image uses BLIP

============================================================
IMPORTANT FILES
============================================================

app.py:
- Flask API
- Serves frontend
- Loads FAISS index and chunks once at startup
- Exposes:
  - GET /health
  - POST /query
  - POST /query/image
  - POST /query/voice
  - POST /query/ocr

static/index.html:
- Current frontend UI
- Blue/white professional dashboard/chatbot hybrid
- Uses relative fetch paths because AMD Jupyter proxy breaks absolute paths
- Must keep these relative API calls:
  - fetch('health')
  - fetch('query')
  - fetch('query/voice')
  - fetch('query/image')
  - fetch('query/ocr')

rag/build_index.py:
- Loads PDFs
- Chunks documents
- Embeds chunks
- Builds FAISS index
- Saves:
  - data/index/manuals.faiss
  - data/index/chunks.json

rag/embedder.py:
- Uses BAAI/bge-small-en-v1.5
- 384 dimensional embeddings
- Normalized embeddings

rag/retriever.py:
- Uses FAISS IndexFlatIP

rag/llm_generator.py:
- Talks to Qwen2.5-7B through vLLM/OpenAI-compatible endpoint
- vLLM base URL is local port 8000

rag/image_analyzer.py:
- Uses Salesforce/blip-image-captioning-base

rag/voice_input.py:
- Uses Whisper base model
- Needs ffmpeg available at /usr/local/bin/ffmpeg

rag/ocr_reader.py:
- Uses EasyOCR, English, gpu=False

rag/answer_generator.py:
- Extractive fallback answer builder
- Filters bad/junk manual fragments

============================================================
IMPORTANT GOTCHAS
============================================================

1. AMD cloud sessions reset packages.
Always reinstall dependencies in fresh sessions.

2. Do not open http://localhost:5000 on laptop.
Use AMD Jupyter proxy:
https://notebooks.amd.com/<instance-id>/proxy/5000/

3. Need two running terminals:
- vLLM on 8000
- Flask on 5000

4. Frontend fetch paths must be relative:
Correct:
fetch('query')

Wrong behind AMD proxy:
fetch('/query')

5. PDF manuals may not be stored in git if .gitignore excludes them.
Download manuals again and rebuild index.

6. FAISS index may be ignored by git.
Run:
python3 rag/build_index.py

7. ffmpeg may not be installed through apt.
Use static binary from johnvansickle.

8. BLIP works better on real equipment photos than synthetic images.

9. EasyOCR works best on clear labels/nameplates.

10. Whisper needs ffmpeg for audio decoding.
